Terminal crash course

1.   || ls ||-----------------------------------(list) ----------------------> content on the folder

2.   || start . ||------------------------------(opener) -----------------> open the folder in file explorer

3.   || ls (folder name) ||-----------------(list inside folder)-------> show list of files and folder inside the typed folder

4.   || pwd ||-------------------------------(printer)-------------------> shows location where the user is right now

5.   || cd ||----------------------------------(open folder)-------------> open folder in git bash or vs code

6.   || cd (folder or file name) ||--------(open folder or file)-----> open folder or file in git bash or vs code

7.|| cd ..|| ----------------------------------(takes back)------------------> it takes back to previous folder only one folder back

8.|| touch (new file) ||-------------------(makes file)--------------> makes files in the location where we are(any type of file can be made from it)

9.|| mkdir (folder name) ||-------------(make folder)------------> makes folders in the location where we are

10. || clear ||-------------------------------(neat and clean)---------> cleans the git bash or terminal

11. || rm (file name) ||-------------------(delete)-------------------> delete the file permanently(end of story)

12. || rm-rf (folder name) ||------------(delete)-------------------> delete the folder permanently(end of story)

13. || ls -a ||--------------------------------(hidden)------------------> show hidden items too and files and folder for sure

14. || ls .. ||---------------------------------(hidden file)-------------> show hidden files in the folder

15. || cd .. ||--------------------------------(folder back)-------------> takes to parent folder if in child folder

3. || git add (fle/fdr name) ||----------(add files)-----------------> add files to staging area

5. || git commit -m "message" ||-----(add all files)-------------> commit all files in staging area

6. || git log ||-----------------------------(all commits)-------------> shows all the commit in the branch or repository
