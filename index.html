Lecture-8(Comparing Changes with Git Diff) ✅✅

21. || git diff ||---------------------------------------------------(list changes)--------------------------> it shows all the unstaged changes made compared to the last commit

22. || git commit -am "commit name" ||-------------------(add commit)--------------------------> it add and commit the changes from the unstaging area to staging area and then commit

23. || git diff HEAD ||------------------------------------------(changes Head)-----------------------> it lists all the unstaged changes and all staged changes from the last commit

24. || git diff --staged ||-----------------------------------------(only staged changes)----------------> it lists only staged changes from the last commit

24. || git diff --cached ||-----------------------------------------(only cached changes)---------------> it lists only staged changes from the last commit

25. || git diff HEAD (filename) ||------------------------------(list particular file)-------------------> it list changes but of the particular filename we write

25. || git diff --staged (filename) ||-----------------------------(list particular file)-------------------> it lists changes but of the particular filename we write

26. || git diff branch1..branch2 ||------------------------------(list branches changes)---------------> it list changes in the branches but the order matter

26. || git diff branch1 branch2 ||------------------------------(list branches changes)---------------> it list changes in the branches but the order matter

27. || git diff commit1..commit2 ||-----------------------------(changes in commit)-----------------> it list changes in the commits

28. || git diff HEAD HEAD~1 ||--------------------------------(lower commit)-----------------> it list changes of lower commit from head to the head commit

Lecture-9(The Ins and Outs of stashing) ✅✅

29. || git stash ||----------------------------------(stash files)----------------------------> it stash the unstaged and staged files in stash so we can change branch and not commit everytime

29. || git stash (save) ||--------------------------(stash files)-----------------------------> it stash the unstaged and staged files in stash so we can change branch and not commit everytime

30. || git stash pop ||----------------------------(empty stash)---------------------------> when its written in terminal all the staged and unstaged files in stash area add to the folder or file

31. || git stash apply ||--------------------------(same stash)---------------------------> when this command is written it add the files to the current folder but it is different from pop it can be used many times and in different branch

Muliple stash can be stored one after the other

32. || git stash list ||------------------------------(list stash)-----------------------------> it helps to show the list of stash stored in the stash stack

33. || git stash apply stash@{number} ||----(use particular stash)----------------> it helps to use the particular number stash in the list of stash

34. || git stash drop stash@{number} ||-----(drop stash)---------------------------> it helps to drop the particular number stash in the list of stash

35. || git stash clear ||----------------------------(drop stash)--------------------------> it clear everything in the stash list
